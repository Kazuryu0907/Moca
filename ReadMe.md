# Moca

- For [GracesBlaze](https://gracesblaze.com/)
- Vtuber用のRocketleagueの大会オーバーレイに特化したアプリケーション
- Rocketleague pluginの[Mugi](https://github.com/Kazuryu0907/Mugi)を使用
- Google DriveとGoogle SpreadSheetでチーム情報，Vtuberモデル画像を管理
